Method -> FunctionStart BlockStmt
FunctionStart -> function Name LParen VarTail
Name -> asdf
VarTail -> RParen | Name VarList RParen
VarList -> EPSILON | , Name VarList
Body -> EPSILON | Statement Body
Statement -> IfStatement | AssignStatement | BlockStmt
IfStatement -> IfToken LParen Expr RParen
Expr -> SumExpr ExprTail
ExprTail -> EPSILON | LessThan SumExpr | GreaterThan SumExpr | EqEq SumExpr
SumExpr -> ProdExpr SumExprTail
SumExprTail -> EPSILON | SumExprList
SumExprList -> EPSILON | Plus SumExpr SumExprList | Minus SumExpr SumExprList
ProdExpr -> Atom ProdExprTail
ProdExprTail -> EPSILON | ProdExprList
ProdExprList -> Times ProdExprList
Atom -> 000 | LParen Expr RParen
AssignStatement -> Name Eq Expr
BlockStmt -> LBrace StmtList RBrace
StmtList -> EPSILON | Statement StmtList
